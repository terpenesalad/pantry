Pantry
a little guide
Pantry
Track items by name, category, quantity, location, stock level and expiry. Search bar has quick-add if something's missing. Swipe left to delete. The ＋ button on each item sends it straight to your shopping list. Categories are draggable — reorder right on the main screen or via the Categories panel.
Shopping List
Out-of-stock items appear here automatically. + Low Stock bulk-adds everything running low. Recipes have a + List button for missing ingredients only. A Maybe section holds treats and specials separately. Tick off as you shop — clearing sends everything back to the pantry.
Recipes
Sorted by what you can actually cook right now — smart matching covers synonyms and substitutes. Filter by Can Make Now, 🌿 In Season, or by cuisine. Cook opens guided mode: tick off ingredients, scale servings, get auto-timers on any timed step. Cooked It deducts from the pantry. Header has a standalone ⏱ Timer and a ⇄ Convert tool (cups, grams, °F, oz and more). Drag cards to reorder.
Bar
Same as Recipes but for cocktails — ingredients, method, glassware, garnish, difficulty, photo. Pantry-aware, guided mode, drag to reorder.
✦ Suggest
AI-powered recipe and cocktail ideas in both tabs. Type any craving, cuisine or occasion — it cross-references your pantry and returns what you can cook now, then what's close. Every result is saved locally so repeat searches are instant and free. Requires an API key — message Kami to get one, then tap ⚙ Key to enter it.

Map
Pin places across Victoria with a name, category and note. Search any address. Tap a pin to view, edit or delete. Hit 🌙 for dark tiles. Pins sync live across devices.
Backup
Top right of Pantry. Downloads everything as a file — restore any time. Do it occasionally.
